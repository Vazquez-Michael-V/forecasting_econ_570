# Econ 570 - Economic Forecasting
Graduate coursework in economic forecasting.

## Projects
- **PS2** - Naive forecasting models applied to S&P 500 monthly data
- **PS3** - Random Walk and Random Walk with Drift models
- **PS4** - OLS and PCA factor models with HAC standard errors
- **Midterm** - Full forecasting evaluation across multiple models and horizons
- **Masters Project** - PCA factor model forecasting the 10-year U.S. Treasury rate (DGS10), benchmarked against a Random Walk using MSFE and the Diebold-Mariano test
